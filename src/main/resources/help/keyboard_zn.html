普通操作

使用鼠标（正常操作）
单击单元格选择该单元格。按下[ctrl]或[shift]同时点击可选择一组单元格。设置单元格/候选数：
  • 双击候选数将其设置到单元格中
  • 如果一个单元格只剩下一个候选数，双击单元格将该候选数设置到单元格中
  • 如果应用了过滤器，双击已过滤的单元格将设置过滤的候选数到单元格中，如果是隐藏单一候选数
  • 按住[ctrl]键双击候选数可以切换候选数
使用鼠标（备选鼠标模式）
如上所述进行区域选择。设置单元格/候选数：
  • 左键单击候选数将其设置到单元格中
  • 如果一个单元格只剩下一个候选数，左键单击单元格的空白处将该候选数设置到单元格中
  • 如果应用了过滤器，左键单击已过滤的单元格的空白处将设置过滤的候选数到单元格中，如果是隐藏单一候选数
  • 右键单击候选数可以取消该候选数，右键单击单元格的空白处可以启用对应的候选数
使用鼠标（着色模式）
在状态栏或活动单元格视图中单击颜色以激活着色模式。单击单元格/候选数可切换该单元格/候选数的所选颜色。按下[shift]键同时单击将使用另一种颜色。 单击背景颜色或按下[esc]键结束着色模式。单击带有“R”的面板结束着色模式并删除所有颜色。
[左], [上], [右], [下]
移动光标（高亮显示的单元格）。
[shift][左], [shift][上], [shift][右], [shift][下]
扩展所选单元格的范围。
[shift][ctrl][左], [shift][ctrl][上], [shift][ctrl][右], [shift][ctrl][下]
如果应用了过滤器，则将光标移动到包含过滤候选数的下一个未解决的单元格。
[ctrl][左], [ctrl][上], [ctrl][右], [ctrl][下]
将光标移动到下一个未解决的单元格。
[home]
将光标移动到最左侧的列。
[end]
将光标移动到最右侧的列。
[ctrl][home]
将光标移动到最顶部的行。
[ctrl][end]
将光标移动到最底部的行。
[1] ... [9]
在高亮显示的单元格中设置值；给定的数独（以黑色显示）无法更改。
[Ctrl][1] ... [Ctrl][9]
切换高亮显示的单元格中的候选数。
[空格]
如果设置了过滤器（见下文），切换过滤的候选数。
[del], [backspace], [0]
删除单元格中的值（给定的数独无法删除）。

Filters

过滤器有两种类型：显示所有允许放置某个数字的单元格（绿色单元格），或显示禁止放置数字的所有单元格（红色单元格）。

[F1] ... [F9]
设置/切换过滤的数字。
[shift][F1] ... [shift][F9]
设置/切换过滤的数字并切换过滤模式（允许/禁止）。
[ctrl][F1] ... [ctrl][F9]
向过滤候选集中添加/删除候选数。如果选择了多个候选数进行过滤，则要么显示包含至少一个过滤候选数的所有单元格（“OR”），要么只显示包含所有过滤候选数的单元格（“AND”）。
[space]
如果过滤器激活且当前单元格已过滤，则在单元格中切换过滤候选数。
[enter]
如果过滤器激活且当前单元格已过滤，则在单元格中设置过滤候选数。
[>], [.]
如果过滤器激活，则设置下一个数字。
[<], [,]
如果过滤器激活，则设置上一个数字。

Coloring

[a]/[A], [b]/[B], [c]/[C], [d]/[D], [e]/[E]
设置/切换所选单元格的背景颜色。
[r]
重置所有单元格的背景。

Miscellaneous

[shift][ctrl]
显示焦点单元格中的所有可能候选数（仅在未设置“选项|显示所有候选数”时）。
[shift][alt]
显示所有单元格中的所有可能候选数（仅在未设置“选项|显示所有候选数”时）。
